# Create Ship Microservice

Things you may want to cover:

* Ruby version
	ruby 2.6.6

* Rails version
	Rails 5.2.6

* Gem install
	`bundle install`

* To run rails server
	rails server  
	run on browser: localhost:3001

# Note 
	This microservice is used for create ship with random name, origin and passengers
	with the delay of 30 to 90 seconds.

# End Point
    GET /ship/new
